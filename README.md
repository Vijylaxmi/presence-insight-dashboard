Presence Insight Dashboard

This repository contains a Power BI dashboard designed to analyze and visualize employee presence and attendance patterns. The dashboard helps organizations track daily presence, missing entries, working hours, and overall attendance behavior using clear and interactive visuals.

Project Overview

The Presence Insight Dashboard provides a data-driven approach to understand:

- Daily employee presence
- In-time and out-time patterns
- Total working hours
- Missing punch-in/punch-out entries
- Monthly attendance trends
- Employee behavior insights

This dashboard enables HR teams, managers, and decision-makers to get a clear view of workforce discipline and productivity.

Key Features

- Interactive Power BI Dashboard  
- Attendance trend visualization  
- KPI indicators for daily/weekly/monthly presence  
- Missing entry analysis  
- Working hours calculation  
- Filters for employee-wise analysis  
- Clean & modern UI design  

Tools & Technologies Used

- Power BI Desktop
- Power Query  
- DAX (Data Analysis Expressions)
- Raw data from Excel/CSV/Database sources

How to Use

1. Download the `.pbix` file
2. Open it in Power BI Desktop
3. Refresh dataset (if required)
4. Explore dashboard visuals & insights
